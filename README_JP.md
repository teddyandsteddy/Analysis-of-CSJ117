# CSJ117の有効性と経済的インパクトの評価
### 傾向スコアマッチングを用いた擬似治験データ解析ポートフォリオ

## 1. 分析の目的
本プロジェクトでは、ノバルティスファーマの治験薬「CSJ117（吸入抗TSLP抗体）」を対象に、経口ステロイド依存や注射剤を忌避する重症喘息患者に対する臨床的有効性と経済的インパクトを評価しました。

## 2. 解析手法と統計的アプローチ
統計学の習熟度と臨床ドメイン知識を示すため、以下のステップで解析を実施しました。

* **データ構成:** Kaggleの「Asthma Disease Dataset」をベースに、フェルミ推定と臨床的知見（FeNO値、アドヒアランス）に基づき加工した擬似治験データを使用。
* **ターゲット層の定義:** 「診断あり・アドヒアランス良好・高FeNO（≥40ppb）」を満たす、最もCSJ117の恩恵を受けるハイリスク群を特定。
* **傾向スコアマッチング (PSM):** 年齢、BMI、喫煙歴等の共変量を用い、ロジスティック回帰により背景因子の偏りを排除。
* **生存時間分析:** * **カプラン・マイヤー法:** 95%信頼区間を伴う非増悪率の可視化。
    * **ログランク検定:** 群間の統計的有意性の検証。
    * **Cox比例ハザードモデル:** ハザード比（HR）による増悪リスク低減効果の定量化。

## 3. 主な解析結果
* **統計的有意性:** ログランク検定により $p < 0.05$ の有意差を確認。
* **リスク低減効果:** Cox回帰により、対照群と比較して顕著なハザード比の低下（[ここにexp(coef)の数値を記入]%のリスク減）を実証。
* **経済的インパクト:** Power BIを用いた感度分析により、注射忌避層の獲得による最大1,200億円規模の市場ポテンシャルを算出。

## 4. 使用ツール
* **Python:** (Lifelines, Scikit-learn, Pandas) 統計モデリング
* **Power BI:** インタラクティブな収益シミュレーションと感度分析
* **Google Colab:** データ前処理から統計検定までの一連のパイプライン
